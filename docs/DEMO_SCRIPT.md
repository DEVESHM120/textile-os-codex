# Demo Script

## Opening

This is a textile workflow automation platform for moving from fabric development data to feasibility review and production label output.

## Flow

1. Open dashboard.
2. Go to Fabric Check.
3. Use the sample cloth card.
4. Run Check.
5. Show deterministic gate, errors, warnings, and rules run.
6. Generate AI Review.
7. Show missing fields, risk summary, correction suggestions, label copy, and technical-sheet text.
8. Go to Label Studio.
9. Generate Excel label workbook.
10. Download artifact from Artifacts.

## Close

The important point is that the same fabric data powers both technical feasibility and documentation, reducing repeated manual checks and copy-paste label work.
