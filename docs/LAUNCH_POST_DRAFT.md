# Launch Post Draft

I built a Textile Workflow Automation Platform for the OpenAI x Outskill AI Builders Hackathon.

It combines two practical textile workflows:

- Fabric feasibility checking for cloth cards, GSM, composition, weave, construction, loom, and width risks.
- Sticker and label generation from the same fabric data.

The MVP turns a cloth card into a feasibility result, AI-assisted technical communication, and a downloadable label workbook in one flow. The feasibility gate is deterministic and SOP-style, while AI helps summarize risks, missing fields, corrections, and production-ready copy.

Built with React, Flask, SQLite, OpenAI structured outputs, and Codex-assisted development.
